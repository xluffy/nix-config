name: grill-me
description: "Interview the user relentlessly until reaching shared understanding, resolving each branch of the decision tree. Use when user wants to stress-test a plan, spec, design, or any document."
---

Interview me relentlessly about this topic until we reach a shared understanding. Walk down each branch of the decision tree, resolving dependencies between decisions one-by-one. For each question, provide your recommended answer.

If the conversation references URLs or external documents, use `yomi read` or `webfetch` to fetch them. Cross-check these with the main topic and use any contradictions, gaps, or hidden assumptions to ask sharper, more pointed questions.

Ask the questions one at a time.

If a question can be answered by exploring the codebase, explore the codebase instead.
