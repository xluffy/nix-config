# Random bookmark nix blog

## Locating Nix Packages

- [nix-index](https://github.com/nix-community/nix-index) — generate a local database and search: `nix run nixpkgs#nix-index -- bin/ip`
- [nix-index-database](https://github.com/nix-community/nix-index-database) — pre-generated database, no local index needed: `nix run nixpkgs#nix-index-database -- bin/ip`
- [Installing a specific version of a package with Nix](https://mplanchard.com/posts/installing-a-specific-version-of-a-package-with-nix.html)
